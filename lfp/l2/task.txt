Профессии - матрица коэффициентов
У каждого факта есть матрица множителей размером с матрицу коэффициентов
    Да -> проф умножаются на коэф
    Нет -> проф умножаются на (1-коэф)
Если при выборе следующего факта ненулевым множителям не соответствует ни одного
    ненулевого множителя професии -> факт пропускается
Вывод = сотрировка коэф профессий по убыванию, каждой профессии соответствует описание
